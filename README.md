# AR-VR POKEMON

## Project Overview
This project demonstrates an **AR-VR Pokémon experience** using **Unity** and **Vuforia Engine**.  
Users can scan Pokémon cards to view interactive 3D Pokémon models in augmented reality.

- **Unity Version:** 2022.x (adjust as per your project)
- **Vuforia Engine Version:** 10.28.4
- **Supported Assets:** FBX, PNG, JPG, MP3, DAE, MAT, ASSET
